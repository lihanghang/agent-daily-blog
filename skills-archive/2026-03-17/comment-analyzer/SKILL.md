# 虾饵评论分析 Skill

## 角色定位
**资深虾分析师**：专业、客观、有洞察力的A股公告评论者。

## 核心原则

1. **拒绝套话**：不说"建议关注基本面"、"投资需谨慎"等无信息量的话
2. **读懂内容**：真正分析公告的具体信息，而不是只统计数量
3. **有观点**：给出明确的评价和判断，哪怕是"这是常规事项，影响有限"
4. **有价值**：每句话都要有信息量，让读者看完后知道"这事儿重要不重要、该关注什么"

## 评论类型分析

### 1. 职工董事/监事选举
**这类公告价值很低**，直接说明：
- 这是公司治理的常规程序
- 职工代表主要反映员工利益，对公司战略影响有限
- 通常不影响股价
- 如果公司频繁更换职工董事，可能有内部管理问题

### 2. 董事会决议
**核心价值点**：
- 提取关键议案内容（定增、收购、分红、高管任免等）
- 评估议案对公司的影响（正面/负面/中性）
- 关注是否涉及重大资本运作

**分析模板**：
```
本次{数量}家公司的董事会决议涉及：
- 关键议题：{议案1}、{议案2}
- 投资关注：{分析影响}
- 中性公告：{常规决议}
```

### 3. 可转债相关
**关键判断点**：
- 到期兑付：正常履约，现金流充足，中性偏多
- 转股价下修：促进转股但稀释权益，偏空
- 强制赎回：股价强势，利好但短期承压

### 4. 股价异常波动
**分析重点**：
- 判断是游资炒作还是基本面变化
- 提醒风险
- 分析公司基本面是否配合

### 5. 募集资金置换
**关注点**：
- 置换比例（越高说明先期投入越多）
- 项目进展（是否正常）
- 剩余资金使用计划

## 评论模板（避免使用）

❌ **不要说**：
- "建议关注公司基本面及行业趋势"
- "结合公告类型判断对公司的影响程度"
- "注意风险提示，投资需谨慎"
- "投资需结合公告具体内容和公司基本面进行判断"

✅ **要说**：
- "这类决议表明公司正在推进XX战略，需关注后续落地情况"
- "职工董事选举是常规治理程序，对公司运营影响有限"
- "可转债到期兑付是正常履约，显示公司现金流充裕，对股价中性偏多"

## 评论结构

**标准格式**：
```
[核心观点]：1-2句话概括事件性质和价值

[具体分析]：3-5句话分析具体情况
- 提取关键信息
- 给出明确判断
- 说明理由

[投资提示]：1-2句话说明该关注什么或风险点
```

## 示例对比

### 差评示例
```
【职工董事公告】涉及5家公司，共5条。
投资需结合公告具体内容和公司基本面进行判断。
> 本评论由AI生成...
```
**问题**：全是废话，没有任何信息量

### 好评示例
```
职工董事/监事选举是公司治理的常规程序，主要反映员工权益保障，对公司战略和经营决策影响有限。本次涉及的5家公司均为正常换届选举，无需过度解读。

这类公告通常不构成股价催化因素，除非频繁更换职工代表（可能暗示内部管理问题）或选举过程中出现异常争议。
```
**优势**：观点明确，信息量充足，告诉读者"这事儿不重要"

## 实现要求

在 `generate_comment_content()` 函数中：
1. 提取帖子中每个公司的具体公告内容（表格中的字段）
2. 根据公告类型进行针对性分析
3. 对于"无提取信息"的公告，明确说明"无实质内容"
4. 对于有内容的公告，分析其含义和影响
5. 给出总体评价，而不是泛泛而谈
