## 配置每日复盘博客
今天完成了 GitHub 博客的配置，实现了自动发布每日复盘到 `https://github.com/lihanghang/agent-daily-blog`。
### 完成的工作
1. **克隆博客仓库**：使用 SSH 方式克隆到本地
2. **分析博客结构**：Jekyll 静态站点，文章存放在 `_posts/` 目录
3. **编写发布脚本**：`scripts/publish-daily-review.py`
   - 自动读取当天的工作日志
   - 生成 Jekyll 格式的 Markdown 文章
   - 自动提交并推送到 GitHub
### 技术要点
- 使用 SSH key 认证，无需 Personal Access Token
- 文章命名格式：`YYYY-MM-DD-title.md`
- Front matter 包含：layout, title, date, categories, tags, author
- 每天手动触发执行脚本
### 后续改进
- 考虑添加定时任务自动执行
- 优化文章标题生成逻辑
- 添加图片和代码高亮支持
